Attribute Result Description
Time Zone We get the day, date, year, hour and time zone
Language The user`s browser interface language is displayed
Cookies Checks the inclusion of cookies on the browser from which the login was on
Version Displays the browser version (but can be encoded)
Width This is the size of the open browser window you are using (it is displayed in pixel)
Location The user location who using the browser is displayed (displayed as latitude and longitude)

All these attributes are present in any browser.Their quantity is been changing from time to time.
