Attribute Result Description
Time Zone We get the day, date, year, hour and time zone
Language The user`s browser interface language is displayed
Cookies Checks the inclusion of cookies on the browser from which the login was on
Version Displays the browser version (but can be encoded)
Size browser window This is the size of the open browser window you are using (it is displayed in pixel)
Screen Resolution This is screen resolution of useing devise
Memory Give the approximate amount of device memory in gigabytes.
Location The user location who using the browser is displayed (displayed as latitude and longitude)
Color Give the bit depth of the color palette for displaying images
Platform Returns for which platform the browser is compiled
Browser We will get code name of the browser
Local Thread It returns the number of logical processors available to run threads on the user's computer
Plugin Return the plugins installed in the application.
Touch Pointer Returns the maximum number of simultaneous touch contact points are supported by the current device.

All these attributes are present in any browser.Their quantity is been changing from time to time.
